Class: Thumb

TokoHPApp.Thumb(id, hpBreakdown, hp, subcat)

new Thumb(id, hpBreakdown, hp, subcat)

Creates a new Thumb. A Thumb represents a thumbnail, aka a piece of artwork. Thumbs are the smallest unit of an HP Journal, and are immutable.
Parameters:
Name Type Description
id string The thumbcode of this Thumb. In the generated HTML, this is used to display the image.
hpBreakdown string A description of why this Thumb has the HP value it does.
hp number The HP value of this Thumb.
subcat string The subcategory of this Thumb. Empty if the user didn't indicate a value for this in their input.
